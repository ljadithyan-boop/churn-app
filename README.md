# 📊 Customer Churn Predictor

A machine learning web app that predicts whether a telecom customer will leave or stay — built with Python and Streamlit.

🔗 **Live App:** [Click here to try it](#) ← replace this with your Streamlit link later

---

## 🤔 What does this app do?

You enter details about a customer — like their contract type, monthly bill, and tenure — and the app instantly predicts whether that customer is likely to **stay** or **leave** the company.

---

## 🧠 How it works

1. Loads real telecom data (IBM Telco Dataset — 7,043 customers)
2. Cleans and prepares the data automatically
3. Trains a Logistic Regression model (81.76% accuracy)
4. Takes your inputs and makes a live prediction

---

## 🛠️ Built With

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

---

## 🚀 Run it locally

```bash
git clone https://github.com/ljadithyan-boop/churn-app.git
```
